## What is Hero
A fast, simple & powerful blog framework, powered by Node.js

### Create a new post

``` bash
$ hexo n "New Post"
```

### Run server

``` bash
$ hexo clean & hexo s
```
### Generate static files and deploy to remote sites
``` bash
$ hexo g -d
```
